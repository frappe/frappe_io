Frappe Framework

Full stack, metadata driven web framework built on Python and Javascript, with a rich admin UI

Documentation

Technology Stack

Frappe comes bundled with an opinionated set of libraries. Here is a non exhaustive set.

  • Programming Languages: Python + Javascript
  • Databases Supported: MariaDB (MySQL)
  • Webserver: Werkzeug
  • Templating: Jinja
  • Job Queues: Python RQ
  • CSS Framework: Bootstrap 3
  • Caching: Redis
  • Realtime: NodeJS + Socketio.js
  • Javascript Packaging: Rollup
  • Webserver (Production): gunicorn + NGINX
  • Process Manager (Production): supervisord
  • Process Manager (Develop): Honcho
  • Command Line: Bench
