# Creando Roles

Antes de crear los Modelos, debemos crear los Roles que van a establecer los permisos en el Modelo. En nuestro ejemplo vamos a tener dos Roles:

1. Librarian
1. Library Member

Para crear un nuevo Role, ir a:

> Setup > Users > Role > New
