# Controls

## frappe.ui.form.make_control
`frappe.ui.form.make_control({ parent, df })`

Makes a frappe control based on `df` properties and appends into `parent`
container.

```js
frappe.ui.form.make_control({
	parent: $wrapper.find('.my-control'),
	df: {
		label: 'Due Date',
		fieldname: 'due_date',
		fieldtype: 'Date'
	},
	render_input: true
})
```
