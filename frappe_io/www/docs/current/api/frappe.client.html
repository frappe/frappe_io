/api/method/frappe.client.get_list

frappe.client.get_list (doctype, fields=None, filters=None, order_by=None, limit_start=None, limit_page_length=20)

Returns a list of records by filters, fields, ordering and limit

Parameters:

  • doctype - DocType of the data to be queried
  • fields - fields to be returned. Default is name
  • filters - filter list by this dict
  • order_by - Order by this fieldname
  • limit_start - Start at this index
  • limit_page_length - Number of records to be returned (default 20)
