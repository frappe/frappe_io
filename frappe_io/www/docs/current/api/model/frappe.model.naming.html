frappe.model.naming._concatenate_autoname (autoname, doc) 

Generate a name by concatenating as many fields as required. This is
called when the doctype's `autoname` field starts with 'concatenate:'. The
field names are separated by a comma. It is also aware of autoname '.####'
format.

frappe.model.naming.make_autoname (key=, doctype=, doc=)

Creates an autoname from the given key:

Autoname rules:

     * The key is separated by '.'
     * '####' represents a series. The string before this part becomes the prefix:
        Example: ABC.#### creates a series ABC0001, ABC0002 etc
     * 'MM' represents the current month
     * 'YY' and 'YYYY' represent the current year

Example:

     * DE/./.YY./.MM./.##### will create a series like
       DE/09/01/0001 where 09 is the year, 01 is the month and 0001 is the series

frappe.model.naming.set_new_name (doc) 

Sets the `name` property for the document based on various rules.

1. If amended doc, set suffix.
2. If `autoname` method is declared, then call it.
3. If `autoname` property is set in the DocType (`meta`), then build it using the `autoname` property.
4. If no rule defined, use hash.

Parameters:

  • doc - Document to be named.
